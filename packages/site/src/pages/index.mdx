# Showcase.js

## React Stories from the Future

Showcase is a React story visualizer and test framework built on top of [Vite](https://github.com/vitejs/vite). Vite has a lot of great features that makes it a great tool for building React components:

- Lightning-fast hot module replacement
- On-demand server start over native ESM
- Optimized bundle for production
